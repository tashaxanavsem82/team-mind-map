# Contributing to Team Mind Map

Thank you for considering contributing to Team Mind Map! We appreciate your help.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes and commit them.
4. Push to your branch and create a pull request.

## Code of Conduct
Please ensure that your contributions follow our Code of Conduct to maintain a welcoming environment for all contributors.
